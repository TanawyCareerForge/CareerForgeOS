# CareerForge OS Code of Conduct

**Version:** 0.1.0-alpha  
**Status:** Draft  
**Purpose:** Establish respectful, inclusive, professional collaboration norms.

## Responsibilities

- Treat contributors and users with dignity.
- Support constructive technical disagreement.
- Reject harassment, discrimination, and abuse.

## Dependencies

- Repository governance
- Security policy

## Inputs

- Community interactions
- Issues and pull requests
- Maintainer reports

## Outputs

- Safe collaboration environment
- Moderation decisions when required

## Validation Rules

- Discussions must focus on ideas, evidence, and project goals.
- Maintainers may moderate conduct that harms community safety.

## Examples

- Acceptable: respectful disagreement with technical rationale.
- Unacceptable: personal attacks, threats, or discriminatory language.

## Test Cases

- Verify moderation decisions reference this policy.
